### Уравнять элементы

**Ограничение времени:** 1 с  

**Ограничение памяти:** 512.0 Мб  

**Ввод:** стандартный ввод или input.txt  

**Вывод:** стандартный вывод или output.txt  

Дана прямоугольная таблица размера $n \times m$ . Каждый элемент в ней — целое число от $1$ до $\min\left\{n, m\right\} − 1$.

За один ход можно выбрать два равных числа, находящихся в одной строке (в одном столбце), и заменить все числа в этой строке (в этом столбце) на это выбранное значение.

Ваша цель — добиться того, чтобы все числа в таблице были одинаковыми. Добейтесь этого не более чем за $2(n + m)$ ходов. Можно доказать, что в данных ограничениях это всегда возможно.

**Обратите внимание:** вам не нужно минимизировать количество выполняемых действий.

### Формат ввода

Первая строка входных данных содержит два целых числа $n$ и $m$ $(3 \leq n, m \leq 100)$.

В каждой из следующих $n$ строк содержится по $m$ целых чисел, лежащих в диапазоне от $1$ до $\min\left\{n, m\right\} − 1$, которые задают значения элементов таблицы в очередной строке.

### Формат вывода

В первой строке выведите количество действий, за которые вы можете сделать все числа равными.

В следующих строках выведите совершаемые действия:

* выведите «R $i$ $j_1$ $j_2$​», если выбираемые равные числа находятся в клетках $(i, j_1)$ и $(i, j_2)$, $j_1$ и $j_2$ должны быть различны; в таком случае все элементы $i$-й строки заменяются на выбранное значение; 
* выведите «C $j$ $i_1$ $i_2$​», если выбираемые равные числа находятся в клетках $(i_1, j)$ и $(i_2, j)$, $i_1$ и $i_2$ должны быть различны; в таком случае все элементы $j$-го столбца заменяются на выбранное значение.

Клетка $(i, j)$ отвечает элементу на пересечении $i$ -й строки и $j$ -го столбца таблицы.

**Пример 1**

**Ввод**

```
3 3
1 1 2
2 2 1
2 2 2
```

**Вывод**

```
3
C 1 2 3
C 2 2 3
C 3 1 3
```

**Пример 2**

**Ввод**

```
4 4
1 2 3 1
2 3 1 3
3 1 2 3
2 3 3 1
```

**Вывод**

```
7
R 3 1 4
C 2 2 4
C 3 1 4
R 2 3 4
R 4 2 3
C 1 2 3
C 4 3 4
```

P. S. Можно вывести любой корректный набор операций, а не только тот, что в примере. Решение представлено в файле `B.cpp` .
