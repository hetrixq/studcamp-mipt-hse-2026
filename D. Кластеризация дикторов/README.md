### Кластеризация дикторов

**Ограничение времени:** 20 с

**Ограничение памяти:** 256.0 Мб

**Ввод:** датасет аудиофайлов
**Вывод:** файл `submission.csv`

### Постановка задачи

Вам предоставлен набор данных, содержащий $N$ аудиозаписей. Эти записи принадлежат $M$ различным дикторам, однако личность диктора для каждой конкретной записи неизвестна.

Ваша задача — распределить аудиофайлы по кластерам на основе идентификации голоса диктора. В идеале каждый кластер должен содержать все записи одного уникального диктора и не содержать записей других дикторов.

### Входные данные

Датасет, содержащий $N$ аудиофайлов в формате .wav. Имена файлов представляют собой случайные UUID (например, `a1b2-c3d4.wav` ).

#### Ограничения

* $N = 1500$
* $M = 50$

Длительность аудио варьируется (обычно 3 – 15 с). Частота дискретизации: 16 кГц (смешанная).

### Формат вывода

Вы должны отправить один CSV-файл с именем `submission.csv` , содержащий два столбца: `filename` и `speaker_id` .

* `filename`: Точное имя аудиофайла, включая расширение.
* `speaker_id`: Произвольная метка (целое число или строка), идентифицирующая кластер.

#### Пример

```csv
filename,speaker_id
5f9a2b.wav,0
1x9c3d.wav,0
8k21lm.wav,1
...
```

*Примечание: Фактические значения speaker_id не имеют значения, важна только группировка, которую они представляют.*

#### Oценка

Решения оцениваются с использованием метрики Adjusted Rand Index (ARI).

ARI — это мера сходства между предсказанными кластерами и истинными (ground truth) кластерами.
